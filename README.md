# ✅ Görev Yönetim Uygulaması

![Uygulama Önizleme](preview.png)

Python ile geliştirilmiş, renkli konsol arayüzüne sahip kapsamlı bir To-Do List uygulaması.

## 🚀 Özellikler
- Görev ekleme, listeleme, düzenleme, silme
- Tamamlandı/bekliyor durum takibi
- Öncelik seviyeleri: Düşük, Orta, Yüksek, Kritik
- Son tarih belirleme
- Görev arama ve sıralama
- İstatistik ekranı
- JSON tabanlı kalıcı veri depolama
- Renkli konsol arayüzü (ANSI)

## 📁 Dosyalar

| Dosya | Açıklama |
|-------|----------|
| `gorev_yonetim.py` | Ana Python uygulaması |
| `gorev_yonetim.ipynb` | Google Colab notebook |

## 🛠️ Kullanım

```bash
python gorev_yonetim.py
```
